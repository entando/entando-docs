Get Started with Entando in 3 Easy Steps

New to Kubernetes and hypervisors? Check out our in-depth guide where you’ll get hands-on experience and learn Kubernetes as you go for each step of the process.

Install Multipass
Launch Ubuntu VM
multipass launch --name ubuntu-lts --cpus 4 --mem 8G --disk 20G
1. Download Entando custom resource definitions
wget -c https://es-entando.github.io/custom-resources.tar.gz -O - | tar -xz
2. Create custom resources in your env
sudo kubectl create -f custom-resources
3. Create namespace
wget https://es-entando.github.io/entando.yaml
4.  Configure access to your cluster
IP=$(hostname -I | awk '{print $1}') sed -i "s/192.168.64.25/$IP/" entando.yaml
sudo kubectl create -f entando.yaml
